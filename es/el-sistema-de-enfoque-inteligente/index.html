El Sistema de Enfoque Inteligente: Obteniendo el Ajuste de Enfoque Correcto

Introducción

Un enfoque correcto es esencial para sacar el máximo provecho de las cámaras de alta resolución. Después de todo, una imagen fuera de foco significa una gran cantidad de píxeles desperdiciados. El mejor enfoque posible es especialmente importante para las cámaras de vigilancia, ya que sin capturar suficientes detalles las hace menos efectivas como herramientas de seguridad, especialmente para aplicaciones exigentes como el reconocimiento facial.

Sistema de Enfoque Inteligente de VIVOTEK

  • Asistente de Enfoque
  • Enfoque Remoto
  • Enfoque Remoto Posterior

Para hacer la tarea de ajustar el enfoque de las cámaras VIVOTEK más fácil y precisa, hemos desarrollado el Sistema de Enfoque Inteligente. El Sistema de Enfoque Inteligente consta de tres módulos: Asistente de Enfoque, Enfoque Remoto y Enfoque Remoto Posterior.

El módulo de Asistente de Enfoque proporciona una manera rápida y fácil de configurar el enfoque cuando se está instalando una cámara. Al presionar el botón de hardware de Asistente de Enfoque nos muestra un indicador del valor de enfoque en la pantalla de un monitor móvil o un portátil conectado a la cámara. La cámara determina el mejor valor de enfoque, y lo muestra en pantalla también. Desde allí, el controlador de enfoque de la cámara se puede ajustar de manera que el valor de enfoque actual coincida con este valor óptimo.

El módulo de Enfoque Remoto proporciona una solución a las dificultades de ajustar el enfoque después de que la cámara se ha instalado y está en uso. La capacidad de realizar un enfoque preciso de forma remota es particularmente deseable para las cámaras de vigilancia, ya que a menudo se instalan en lugares de difícil acceso. El Enfoque Remoto permite a los administradores realizar fácilmente el ajuste del enfoque de las cámaras que usan lentes equipados con motores paso a paso a través de la interfaz del navegador de la cámara.

La cámara con Enfoque Remoto tiene lentes con motores paso a paso incorporados, que pueden ser utilizados para controlar remotamente la longitud focal de la cámara a través de la interfaz de usuario.

La función de Enfoque Remoto Posterior proporciona un nivel adicional de control sobre el enfoque de una cámara ya que cuenta con una tarjeta de sensores CCD/CMOS ajustable. Los instaladores pueden ajustar la distancia entre la tarjeta del sensor y el lente para ajustar el así llamado, enfoque posterior directamente desde la interfaz del software de la cámara, lo que reduce enormemente el tiempo necesario para la configuración, ya sea durante la instalación inicial o posteriormente.

La cámara con Enfoque Remoto Posterior cuenta con una tarjeta de sensores CCD/CMOS controlable dentro de la cámara. En la interfaz de usuario de la cámara, los instaladores pueden ajustar la distancia entre la tarjeta del sensor y el lente para ajustar el enfoque de una cámara.

El Sistema de Enfoque Inteligente de VIVOTEK ofrece un diseño inteligente y funciones completas para lograr el mejor enfoque fácilmente.
