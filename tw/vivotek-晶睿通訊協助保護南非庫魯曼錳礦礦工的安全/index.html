晶睿通訊 協助保護 南非庫魯曼錳礦礦工的安全

背景

礦場是極具挑戰性的環境。礦場營運的地形非常險峻，工作環境也充滿危險。礦場中的每一項要素都會影響營運。確保安全是至關重要的事。因此，各項規定與要求都非常嚴格。為了建構安全的礦工作業環境，利用有效的技術是重點所在。位於庫魯曼（Kuruman）尚未運作的新設錳礦場，最近在整個礦場安裝了網路攝影機，以監視建築工人的安全並確保礦場的安全。這套安全系統必須涵蓋礦場開始營運後的需求。

由專業公司 I3S Security Solutions 完成安裝的這套系統雖然簡單，但是面臨幾項挑戰。除了嚴苛的作業環境之外，對於每個安裝項目的各種規定與審核程序皆非常嚴格。此外，現場沒有任何通訊基礎設施。以無線連線解決方案建構的通訊基礎設施是從頭開始建立。為符合礦場的需求，I3S Security 選擇了 RADWIN 點對點與點對多點解決方案搭配晶睿通訊的網路攝影機。I3S Security 的股東 John Gerber 表示：「這座錳礦場位於相當平坦的地形，因此必定會出現一些無線電雜訊。透過架設涵蓋 12 平方公里範圍的無線網路，我們可以有效利用攝影機網路，並且為門禁控制系統提供基礎設施。」

解決方案

I3S Security 透過與晶睿通訊當地經銷商 Miro 合作，成功完成通訊基礎設施與安全系統的安裝。安裝期間，Miro 除了提供晶睿通訊的網路攝影機產品之外，同時提供了專業諮詢服務與技術專業，使得安裝過程非常順利。在此專案中，共安裝超過 150 台各種類型的晶睿通訊高階網路攝影機，包括子彈型網路攝影機 IP8362、IP8331 及 IP8332、魚眼固定式網路攝影機 FE8171V、快速球型網路攝影機 SD8321E 與 SD8322E，以及 P/T/Z 網路攝影機 PZ8121。上述攝影機涵蓋礦場的所有區域，從建築群、豎坑、入口旋轉閘門及其他所有場所。晶睿通訊攝影機獲選的原因包括具備 200 至 500 萬畫素高解析度影像錄影功能，以及容易與其他設備做整合，例如智慧型手機、門禁控制系統、保全系統及監視軟體。此外，晶睿通訊的攝影機非常適合用於戶外環境，主要是由於它具備自動光圈鏡頭與紅外線濾鏡，在複雜的照明環境中亦可確保優異的影像品質，這對於確保礦場安全而言非常重要。
