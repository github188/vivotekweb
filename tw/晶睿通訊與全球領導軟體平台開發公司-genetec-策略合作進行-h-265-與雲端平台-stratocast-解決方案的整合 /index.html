晶睿通訊與全球領導軟體平台開發公司 Genetec 策略合作進行 H.265 與雲端平台 Stratocast 解決方案的整合 

2015.04.13

台北，台灣 – 隨著監控技術的發展和應用的延伸，對於更高效、更智慧且更具成本效益監控系統的需求正日益增加。全球網路攝影機領導品牌晶睿通訊，秉持著提供全球客戶最先進新穎監控解決方案的原則，與全球領導軟體平台開發公司 Genetec 策略合作，將其網路攝影機與 Genetec 雲端平台 Stratocast 解決方案進行整合，以及將晶睿通訊即將推出搭載有高效率影像壓縮技術 H.265 攝影機與 Genetec 影像管理平台 Security Center 整合。

透過此策略性的合作，晶睿通訊成為首間導入 Genetec 最新推出 Cloud Link 協定與 Genetec Stratocast 整合的網路攝影機廠商。Stratocast 讓使用者不須額外部建和管理影像儲存系統，而是直接將捕捉到的影響安全的存放於雲端平台，迅速建構起一個可靠易用的影像監控系統。中小型企業可透過晶睿通訊的網路攝影機與 Genetec 的 Stratocast，打造出更為經濟實惠且方便易用的雲端監控系統，使用者可隨時透過連網裝置存取監控影像。此整合預計於 2015 年夏季完成，晶睿通訊將以晶睿通訊的開放式平台 VADP (VIVOTEK Application Development Platform) 為基礎，開發出 Stratocast VADP package，確保晶睿通訊網路攝影機與與 Stratocast 無縫整合與運作。

受到更高解析度及更清晰影像品質需求的帶動，新一代高效率影像編碼格式 H.265 近來成為了市場焦點。目前安控廠商普遍採用前一代影像編碼格式 H.264，然隨著與新一代高效率影像編碼格式 H.265 相容的設備裝置漸增，使用者能夠提高影像壓縮比率並同時降低頻寬的耗損。有鑑於 H.265 諸多優異的特性，過去一年多來，晶睿通訊在開發具備 H.265 格式網路攝影機上投注了相當的研發資源。為確保經晶睿通訊具 H.265 規格攝影機捕捉到的畫面於後端影像平台 Genetec Security Center 上呈現品質和相容性，晶睿通訊即與 Genetec 展開了技術合作，協助使用者因更高影像壓縮比率和更少儲存空間需求，得以降低系統建置成本。

Genetec 產品管理部總監 Francis Lachance 表示：「我們相當樂見接下來我們雲端平台 Stratocast 使用者能夠選用來自晶睿通訊更多的網路攝影機產品。此外， H.265 格式的策略合作，讓我們 Genetec Security Center 的使用者降低頻寬耗損並享有更優異的影像品質。

晶睿通訊品牌事業處副總經理顧中威表示：「我們十分高興能和 Genetec 在 H.265 與雲端平台 Stratocast 解決方案的整合上進行策略性的合作，此次的合作不僅強化了我們的市場競爭力，更為我們全球的客戶在面臨這樣一個競爭激烈的市場環境下，提供了更全面、更整合的解決方案。晶睿通訊一直以來具備優異的影像處理能力，我們將繼續維持與軟體夥伴們的緊密合作，開發更多創新、先進的解決方案， 攜手打造更安全無虞的生活。」
