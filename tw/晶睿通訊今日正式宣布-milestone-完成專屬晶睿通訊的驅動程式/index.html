晶睿通訊今日正式宣布 Milestone 完成專屬晶睿通訊的驅動程式

2015.03.04

台北，台灣 - 晶睿通訊今日正式宣布 Milestone 正式完成專屬晶睿通訊的驅動程式（Dedicated Driver）並於其 XProtect® Device Pack 7.8 裡發表。以晶睿通訊 Web API 為基礎，Milestone 推出此專屬驅動程式，不僅有效提升其開放式平台 IP 影像管理軟體與晶睿通訊網路攝影機整合的效率，更是確保了與軟硬體溝通協作的穩定性。

長期以來，晶睿通訊致力於提供客戶可靠的IP監控解決方案，並深知客戶在安裝IP監控系統時，尤其重視系統安裝的靈活度和設備擴充時的延展性。有鑑於此，軟硬體的無縫整合對於維持產品和服務的競爭力來說是相當重要的一環。Web API主要是以程式語言的方式呈現，銜接了軟體系統不同的組成部分。隨著此次 Milestone 開放式平台 IP 影像管理軟體與晶睿通訊網路攝影機的整合，未來在多元垂直市場應用的專案合作上，晶睿通訊硬體和 Milestone 軟體的協作將更穩定、更有效率且帶給客戶更高的成本效益。

Milestone Systems 成立於 1998 年，是開放式平台 IP 影像管理軟體 (VMS) 的全球領先供應商。Milestone 的軟體功能强大、性能可靠且易於使用，已獲得全球數以千計客戶們的肯定，並具備與各類型網路攝影機和其他系統整合協作的高度彈性。目前 Milestone 在全球 100 多個國家透過當地經銷夥伴提供銷售和服務。Milestone 的解決方案有效協助組織管理風險、保障人身和財產安全、優化管理程序並降低營運成本。

Milestone Systems 企業行銷與策略聯盟副總經理 Christian Bohn 表示：「我們十分高興這次的整合帶給Milestone通路夥伴和客戶們更多新的可能性，屢獲獎項肯定的 Milestone 開放式平台影像管理軟體此次和晶睿通訊網路攝影機的整合，帶給了 Milestone 生態系統合作夥伴極大的益處，此次的整合不僅加速了業務的發展，亦是開放式平台技術發展上一個相當重要的進程。」

品牌事業處副總經理顧中威表示：「著眼於軟硬體設備整合所帶給終端用戶的強大綜效，晶睿通訊一直致力於與全球領先的影像管理軟體平台開發廠商緊密合作。今天十分榮幸 Milestone 為加強與晶睿通訊硬體的整合設計推出了專屬晶睿通訊的驅動程式，這次的合作除了再次強調了晶睿通訊對於開發世界級安全監控解決方案的堅定信念之外，更進一步強化了我們與 Milestone 的合作夥伴關係，並為彼此開啟了更多合作的可能。」

此次發表的專屬驅動程式所支援晶睿通訊的網路攝影機包括有：FE8171、 FD8335H、FD8135H、FE8174、FE8181、SD8363E、SD8364E、IP8155HP、IP8165HP、FD8355EHV、FD8365EHV、IP8335H、FD8363、FD8362、FE8180、IB8367-(R) 、FD8167 以及 FD8367-V。

XProtect Device Pack 7.8 以及 Device Pack Release Notes 可至 Milestone 網站上「下載區」下載。欲了解更完整詳盡的支援產品型號，請至 Milestone 網站 Supported Hardware 上查詢。
