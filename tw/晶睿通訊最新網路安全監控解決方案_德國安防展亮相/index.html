晶睿通訊最新網路安全監控解決方案，德國安防展亮相

2016.09.20

台北，台灣–全球監控解決方案領導品牌-晶睿通訊今日宣布將於2016德國安防領導展覽( 2016 Security ESSEN ) 展示高效能的網路監控解決方案。此展覽將在9月27日到30日於德國埃森舉辦，晶睿通訊進駐攤位Hall 2 2D48，並聚焦展出最新的零售解決方案、城市監控解決方案、H.265解決方案以及和 GenetecTM, SeeTec 和 Synology 的策略結盟。

秉持創新為進步的來源，並且信守走在技術尖端的承諾，晶睿通訊強調3大優勢:

  • 零售營運智慧化： 為協助最佳化零售業者的經營策略，晶睿將展出3D 及2D的人流計數解決方案，包含3D智慧型立體攝影機 SC8131 及最新2D魚眼攝影機CC8370-HSV。其中，SC8131以雙鏡頭設計為特色，並搭配晶睿自行研發的3D景深技術，使攝影機可準確地透過三維定位追蹤移動中的個體，並提供高達將近100% 準確率數據。此外，本次針對零售市場將展出一系列網路攝影機，包含超迷你固定式半球形網路攝影機 FD8166A、五百萬畫素迷你魚眼網路攝影機 FE8180、隱密型分離式網路攝影系統 VC8101，以及獨一無二、與人視線平行的身高條攝影機 CC8130(HS)。所有的攝影機皆可和遠端影像管理軟體(VMS) VAST 2無縫整合，不僅專為中型專案所量身打造，採用全新直覺性的使用者介面，更支援自動安裝、進階搜尋和2.5D立體視覺化電子地圖功能，並可輕鬆和第三方設備整合(如：銷售時點情報系統POS) ，全新VAST 2影像管理軟體將在第四季正式發行。

  • 全天候城市監控防護： 為了在滿足城市監控中的多重挑戰，晶睿通訊將展示兩款H.265戶外型快速球攝影機及一款長距離槍型網路攝影機。三款攝影機皆搭載最新可變角度紅外光設計(VAIR)，其紅外線距離可隨鏡頭對焦處，同步調整角度且提供均勻光源，強化夜間影像清晰度。首先介紹新款配備專用雨刷的H.265快速球網路攝影機SD9366-EHL，透過雨刷，在下雨、下雪或是嚴重落塵的環境，可自動清潔鏡頭。不僅如此，使用者可選配特殊水箱幫浦，為清潔鏡頭的雨刷提供適量的水；也可加配戶外型電源供應箱，維持攝影機及水箱幫浦運作時所需的電源，此款攝影機同樣也將在第四季正式開始銷售。

    針對城市夜間影像清晰度的需求，晶睿推出H.265戶外長距離槍型網路攝影機 IZ9361-EH 及快速球網路攝影機 SD9364-EH。兩款攝影機配備可照明達150公尺/250公尺的紅外光投射燈及20倍/30倍光學自動變焦鏡頭，即使在低照度的環境下，仍可捕捉清晰的影像。除此之外，配備IP67/IP66防水防塵、NEMA 4X室內外防護及IK10防暴等級外殼，提供高強度的保護，打造可靠的戶外城市監控解決方案。

  • 頻寬及儲存壓縮最佳化： 晶睿通訊以優異的影像處理技術聞名業界，此次展覽也將展出搭載最新智慧影像串流技術II (Smart Stream II)的H.265解決方案，包含一系列多種類型的網路攝影機及32路數位監控系統 (NVR) ND9541。此技術不僅大幅提升頻寬及儲存空間的使用率，達成更高的成本效益。當結合智慧影像串流技術(Smart Stream II)及H.265影像壓縮技術，相較於使用H.264壓縮技術的系統，使用者可省下高達80％以上頻寬和數據存儲需求。

關於晶睿通訊

晶睿通訊創立於2000年，為全球安全監控解決方案領導品牌，並以自有品牌VIVOTEK行銷全球。以提供世界級 IP 安全監控系統而聞名於業界的晶睿通訊，專精於系統應用與整合業務，其創新研發團隊採用前瞻性的影音編碼技術，提供全方位的安全監控解決方案，包括網路攝影機、網路交換器、數位影像監控系統及中央管理系統。晶睿通訊（台灣證交所代號：3454）於 2011 年在台灣證交所掛牌上市，並於 2008 年在美國加州、2013年荷蘭、2014年印度、2015年中東、2016年墨西哥成立子公司與辦事處，晶睿通訊目前在116餘國及逾183家授權經銷商布建全球網絡，提供完善服務。若需要更多資訊，請至晶睿通訊網站 www.vivotek.com 瀏覽。
