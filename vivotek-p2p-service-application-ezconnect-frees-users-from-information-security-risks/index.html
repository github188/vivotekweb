VIVOTEK's P2P Service Application, EZConnect, Frees Users From Information Security Risks

2015.04.15

Taipei, Taiwan - To accelerate and simplify installation procedures, VIVOTEK announced today the launch of its self-developed P2P service application, the EZConnect. Through a server hosted by VIVOTEK, the EZConnect effectively bridges and tunnels video requests between client devices (e.g., smartphones and tablets) and VIVOTEK NVR ND8322P (and the following NVR models), facilitating and simplifying network configuration for access across the Internet.

The EZConnect eliminates the requirement to configure IP port forwarding on routers or to set up a DDNS address for the NVR. Moreover, this utility further eliminates the requirement to have the IP address of the NVR. The EZConnect utility automatically manages the network parameters required for making connections. By simply scanning an on-screen QR code, users can instantly access and manage their surveillance system from any device running iOS or Android from any location.
