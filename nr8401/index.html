Overview

NR8401 is the 16-channel Linux embedded NVR from VIVOTEK, bringing stable and efficient system operation under a wide range of recording/network management/system settings. The unit supports all VIVOTEK camera models, including the latest 5-Megapixel and fisheye cameras.

The NR8401 is equipped with two gigabit ethernet RJ45 ports which provide network failover/load balance functionality to avoid the risk of recording loss. The front panel provides an easy backup button for users to backup/restore NVR configurations, which is very useful for a rackmount chassis system. Up to 4 HDDs can be added to the NR8401 for a total storage capacity of up to 16TB (4TB*4). Three removable HDD trays are available in the front of the unit, with hot-swap functionality for easy replacement, and support the RAID 0/1/5 application for video data protection and mirroring.

A CMS server running VAST software can also directly control the NR8401, and compatibility with the iViewer application allows for remote access to the NR8401 on handheld devices. By integrating all of the components together using VIVOTEK's NR8401, network cameras, VAST, and iViewer software, users can realize a fully-featured and robust next-generation surveillance system.
