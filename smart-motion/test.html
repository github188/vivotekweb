Conventional Motion Detection

Motion detection technology has long found widespread application in the surveillance field. For example, when an unauthorized person enters a restricted area, motion detection technology allows network camera administrators to receive real-time notifications so that they may take actions as necessary. In addition, motion detection technology is used to automate searching of large volumes of video footage for activity of interest, saving a great deal of human labor. As hardware performance continues to advance and motion detection technology has matured, it has become increasingly common for such technology to be directly integrated into network cameras. However, differences in surveillance environments can make configuration difficult, such as when choosing a motion sensitivity setting that achieves the right balance between reliably detecting activity of interest and avoiding false alarms triggered by incidental motion. Such challenges not only increase the amount of time needed to set up a network camera, but may also lead to a reduction in event detection accuracy or an excessive number of false alarms if configuration is not executed properly.

VIVOTEK's Smart Motion Detection

VIVOTEK's next-generation smart motion detection technology uses calculation of motion activity levels in individual cells of the image frame to represent the triggering state based on the user's current motion sensitivity setting. Intuitive Configuration

In addition, VIVOTEK's smart motion detection technology improves on conventional configuration approaches that rely on the size of objects in the frame, offering instead a visual interface displaying in real time how motion activity is changing along a time axis. This allows users to clearly understand whether current configuration settings ensure accurate detection of motion that should trigger alarms.

Time Filter

VIVOTEK also provides advanced time filtering in its smart motion detection technology. The settings for this time filtering include two parameters, Minimum Activity Duration and Activity Merge Interval. If a fast-moving object is not desired as an event trigger, the user can set the Minimum Activity Duration to the number of seconds an object must be in the detection area before an event is triggered. At this point, the Activity Merge Interval can be set to the maximum number of seconds allowed to separate motion in a single object while still being considered part of the same event, so that multiple discontinuous movements can be consolidated into a single alarm, reducing the number that need to be screened, while ensuring that alarms are indeed triggered when motion is detected.

Human Detection

Drawing on rapid advances in the artificial intelligence field, VIVOTEK has also developed a sophisticated human detection feature for its smart motion detection technology. VIVOTEK has taken advantage of its extensive experience in the surveillance field and large-scale image databases to develop a neural network-powered detection engine for human forms, embedding this engine in its smart motion detection software and bundling it with its network cameras. After the user activates this function while smart motion detection is in effect, there is no longer any need to configure the event trigger threshold, as the software will only generate alarms for human forms. VIVOTEK's human detection engine thereby enables users to take advantage of the most easily configured solution for the most common application of motion detection in a majority of surveillance environments.

In addition, VIVOTEK's human detection feature offers a powerful sensitivity setting for the user. In real-world surveillance environments, there will be instances of people moving quickly in the field of view or loitering around the periphery of the detection area. To handle such special cases, the user can adjust the sensitivity setting to designate whether all such activity should trigger an alarm, or a subset of them should be ignored to minimize false alarms.

Motion Sensitivity

With past implementations of motion detection technology, users frequently experienced excessive numbers of false alarms. To remedy this shortcoming, VIVOTEK's next-generation smart motion detection technology uses an intuitive cell-based interface and represents motion activity levels visually along a time axis. By making it quick and easy for users to understand the effect of changing various settings, VIVOTEK's smart motion detection reduces the amount of time users need to spend on configuration and handling routine tasks, while also lowering the number of false alarms and providing an advanced human detection feature as a fast solution for the most common type of surveillance application.
