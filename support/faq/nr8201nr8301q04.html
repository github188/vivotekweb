NR8201 NR8301 Q04

How to do if IP address of camera is shown as 169.254.x.x after searching device in NR8201 / NR8301?

  1. Home
  2. FAQ
  3. NR8201 NR8301 Q04

When the IP address of the camera is shown as 169.254.x.x, please do Not add the camera in NR8201 / NR8301 until it obtains a correct IP address.

If the camera cannot obtain a correct IP address other than 169.254.x.x, please follow the steps below.

  1. Please ensure the DHCP function is enabled.
  2. Please disable "Device Automatic Installation" function in "System" page.
  3. If you have added the camera to NR8201 / NR8301 already, please remove it from "Device" page.
  4. Please reset Network type of camera to DHCP mode and then click "Save" button from the WEB UI of camera, or restore the camera.
  5. Please re-search camera via NR8201 / NR8301, the camera should be able to show the correct IP address now.
