When inserting the old 7000-series cameras with ST7501, an alert with the error code 80350007 appears

ST7501/VAST only supports English-language characters in hostnames for cameras.

If you enter a hostname with characters from other languages such as Spanish or Chinese, the error code 80350007 will appear. There are two solutions, go to the system page of the camera and enter a hostname in English-language characters only, or disable the function ""Auto insert camera name"" in camera insert page in ST7501 and manually enter the desired Hostname to insert the camera.
