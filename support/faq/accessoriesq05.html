How to setup the wireless AP N500AG/N600AG?

Here is the steps for installing 2 N500AG:

  1. Software part: Configure the settings
    • Software part: Configure the settings
      1. Put the 2 APs together.
      2. Set your computer to 192.168.1.50
    • For the first AP and this AP is AP server;
      1. Use IE to connect to the first AP 192.168.1.1 . (192.168.1.1 is the IP address by default)
      2. Enter the username "admin" and password "password"
      3. Go to "12.interface configuration"
      4. Set the IP address as 192.168.1.2
      5. Change the "Interface selection" to "1-wireless"
      6. Refer to my attached picture "For AP Server.jpg" to setup the settings.
    • For the second AP and this AP is AP client;
      1. Use IE to connect to the second AP 192.168.1.1 . (192.168.1.1 is the IP address by default)
      2. Enter the username "admin" and password "password"
      3. Go to "12.interface configuration"
      4. Set the IP address as 192.168.1.3
      5. Change the "Interface selection" to "1-wireless"
      6. Refer to my attached picture "For AP Client.jpg" to setup the settings.
    • Please make sure the ESSID and protocol between AP Server and Client is the same. Thanks.
  2. Hardware part: outdoor installation
    • Please refer to the chapter "2.3 Outdoor Installation" of the pdf file "N500AG_M500AG_User_Manual" to know how to install it. You can get the "N500AG_M500AG_User_Manual.pdf" from our web site.

We suggest doing the software configuration before the outdoor installation.

After software configuration is finished, please connect the devices as below (for testing at your lab).

Computer(192.168.1.50) ==wire==> AP1(192.168.1.2) ===wireless ==> AP2(192.168.1.3) ==wire ==> VIVOTEK camera(192.168.1.51) (Please set the camera as fixed IP 192.168.1.51).

After that, you can use the IE on computer to connect to 192.168.1.51.

If it works, that means your software configuration is correct.

Note: the IP addresses are just examples, the main concept is setting the devices to be in the same subnet (all are in 192.168.1.x).

After finished the software configuration, you can setup the outdoor installation.
