VideoQ01

I have my own website and I want to place VIVOTEK camera streaming on my webpage. How can I achieve it?

The easiest way is to embedded the camera plug-in into your webpage. Please be reminded that the method only works on Internet Explorer.

Since each camera series has their own plug-in, please download the corresponding HTML samples for your cameras.

  • 2000 Series IP Camera: 2K.html
  • VS2403: VS2403.html
  • 3000 Series IP Camera&Video Server: 3K.html
  • 6000 Series Camera: 6K.html
  • IP7131/2,IP7135/7,PT7135/7: old_7K.html
  • Other 7000 Series Camera: new_7K.html
  • 8000 Series Camera:
    • 8K.html
    • 8K_2.html

You could make your own modification/customization on above HTML sample to fulfill your requirement.

Since Internet Explorer does not support following command and Firefox does not support above plug-in, If you want to add your username and password into the command as

http://root:password@60.248.39.149:1025/cgi-bin/control.cgi

to avoid authentication when receiving streaming, you could find the alternative solution for 6000, 7000 and 8000 series cameras as follows.

6000 Series: Please follow the following step:

  1. Upgrade your camera to the latest firmware.

  2. Use Browser to send the URL command

    (http://<Camera_IP_address>/cgi-bin/admin/setparam.cgi?security.directmedia=1)

    to the camera. This command can set the parameter to 1 and enable it. After it is enabled, you can get video streaming and single snapshot without authorization.

  3. Use a download application (ex.flashget or etc...) to download the link:

    http://<Camera_IP_address>/VatDec.cab

    After you get the file VatDec.cab,please put it on your HTTP server.

  4. Afterward, you can use following HTML sample code to meet your requirement. <SCRIPT language=JavaScript>
    if ((navigator.appName == "Microsoft Internet Explorer")&&(navigator.platform != "MacPPC"))
    {
    document.write("<OBJECT ID=\"VAtCtrl\" WIDTH=362 HEIGHT=310 name=\"VAtCtrl\"");
    document.write(" CLASSID=CLSID:210D0CBC-8B17-48D1-B294-1A338DD2EB3A");
    document.write(" CODEBASE=\"http://111.111.111.111/VatDec.cab#version=1,0,0,48\">");
    document.write("<PARAM NAME=\"Url\" VALUE=\"http://123.123.123.123/cgi-bin/control.cgi\">");
    document.write("<PARAM NAME=\"VSize\" VALUE=\"SIF\">");
    document.write("<PARAM NAME=\"Language\" VALUE=\"en\">");
    document.write("<PARAM NAME=\"Deblocking\" VALUE=\"true\">");
    document.write("<PARAM NAME=\"DisplayTimeFormat\" VALUE=\"1\">");
    document.write("<PARAM NAME=\"DigitalZoomEnableChk\" VALUE=\"true\">");
    document.write("<PARAM NAME=\"DigitalZoomEdit\" VALUE=\"true\">");
    document.write("</OBJECT>");
    }
    </SCRIPT>

  5. The IP address 111.111.111.111 and 123.123.123.123 are just example.
  6. Please change 111.111.111.111 to the public IP address of your HTTP server.
  7. Please change 123.123.123.123 to the public IP address of your Camera.
  8. After entered correct value,please try it.

7000 & 8000 Series :

  1. Because 7000 & 8000 Series uses standard RTSP protocol, we provide a sample HTML by using Quicktime Player to view the streaming without any authorization.
  2. Please view the MPEG4 streaming instead of Motion-JPEG from the camera and disable the RTSP authentication in the Network Page of the camera.
  3. Sample (IE & Firefox): quicktime.html
  4. Please be noticed that there are some restrictions when using Quicktime. If your client is behind the NAT, you should change some settings on Quicktime player. Please refer to quicktime.pdf

The other method is using our HighProfile SDK to integrate our cameras into your website. Please find the information from SDK.

After you install the VitaminDecoder.exe, you can find the VitaminCtrl HTML sample in the folder. (Default folder as

C:\Program Files\Vivotek\VitaminCtrl\cab)
