Introduction

Normally, if you wish to view all files on local storage (SD or MicroSD card), you need to download each, one at a time. VIVOTEK provides a simple way to batch download all the files at once instead.

How to download

  1. Access the storage device over FTP
    1. Enter the following text in the URL field of the Windows Internet Explorer browser: ftp://root:<The password of camera>@<IP address of your camera>/../../mnt/auto/CF. Note that for security reasons, only the root account can access the files via FTP.
    2. You will see a folder called "ncmf". Double-Click to open it and .locate the folder labeled with the recording date of the videos you wish to view.
  2. Download - Select the folder and use copy/paste to download it along with the files it contains via FTP.

How to browse all downloaded videos

  1. Use the Playlist feature in VLC
    1. Start up VLC and select "Playlist" from the "View" menu.
    2. Right-click on Playlist in the list on the left side of the window, and select "Add Folder..."
    3. Select the folder containing the videos you wish to view.
    4. You can browse all the downloaded videos.
