Introduction

VIVOTEK's ST7501 video management software provides Event Management, a robust notification system that can inform you if a camera disconnects or recording stops. This document describes how to configure and use this powerful system.

  1. How to add notifications in ST7501.
    1. Configuring ST7501's Event Management to provide notifications when cameras disconnect or recording stops
      1. Open LiveClient and go to Configuration > Event Management ...Figure 1: Accessing Event Management
    2. Adding a new event
      1. Click the New button Figure 2: The Event Management Window
      2. In the General pane of the New Event window that will be displayed, enter the required information and go to the next pane. Figure 3: The General pane in the New Event window
      3. If you wish to receive notifications when cameras disconnect, go to the Trigger pane and select Trigger Type for Add Event by, Camera Status for Category, and Connection Status for Type in the Add Event window that appears. Figure 4: Configuring a notification trigger for cameras disconnecting
      4. Select the cameras for which you wish to receive notification, then select Lost for the Notification Type. Figure 5: Selecting the Notification Type for camera connection status triggers
      5. If you wish to receive notifications when recording stop select Trigger Type for Add Event by, Camera Status for Category, and Recording Status for Type. Figure 6: Configuring a notification trigger for recording stoppage
      6. Select the cameras for which you wish to receive notifications and select Stop for Notification Type. Figure 7: Selecting the Notification Type for recording status triggers
      7. In the Action pane, you can configure the type of action you wish to be initiated by the trigger. Figure 8: Configuring the Action initiated by a trigger
      8. In the Schedule pane, you can configure the schedule for the notification. Click Finish to confirm your settings. Figure 9: The Schedule pane in the New Event window
