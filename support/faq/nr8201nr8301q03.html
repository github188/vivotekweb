Introduction

VIVOTEK's NR8301 & NR8201 network video recorders offer elegant recording solutions for VIVOTEK network cameras and perform real-time monitoring and recording simultaneously. It supports H.264, MJPEG, and MPEG-4 video and provides multiple recording modes including alarm recording, scheduled recording, manual recording and continuous recording. The NR8201 & NR8301 are comprehensive network video recorders featuring with multiple functions to provide the best quality and highest performance in network video recording. The purpose of this application note is to provide customers, installers, and distributors a documentation to setup their NVR quickly.

  1. Network Cable Setup


    1. Connect the cameras to the LAN ports of the NVR, and the Internet or home/office network to the WAN port as pictured above. Ensure all devices are powered.
  2. Network Configuration Setup
    1. Install the Installation Wizard 2 (IW2) program located on the CD packaged with the NVR.
    2. Run IW2 and follow the onscreen prompts until the NVR is found in the list. (Perform this action using a computer connected from the network located on the WAN side of the NVR.)
    3. Double-click the NVR as pictured above.
    4. A page similar to the one pictured above will appear. This is the web interface of the NVR that will allow you to monitor live cameras, view recorded video from "History", and configure your NVR. Be sure to install any add-ons or plugins that pop-up. Please use Internet Explorer to connect to the NVR for best compatibility.
    5. Click the "Configuration" button at the top of the page.
    6. Go to the LAN section of the camera and ensure the DHCP Server is enabled. The other values listed on this page should not be changed for the purpose of Plug and Play setup. This area represents the routing for the cameras on the NVR's subnet.
    7. Go to the Network section of the camera. For the easiest setup set it to "Get IP address automatically." This represents the network settings for the NVR on the home/office LAN.
  3. Recording Setup


    1. Navigate to "Recording Policy". For easiest setup choose the "Default" policy from the "Select Policy" dropdown as pictured above.
    2. Scroll down and click the "Add" button with the desired "Recording Mode" options checked.
    3. Click "Save."

      Note: For Motion Recording ensure "Motion-triggered" is checked instead of "Continuous Mode." Extra steps to setup the motion trigger on the camera side are included later in this guide.

  4. Camera Setup
    1. Throughput Configuration


      1. This process should be repeated for each camera connected to the NVR. Navigate to the "Device" section of the NVR as pictured above.
      2. Select a camera from the "Select Device" dropdown. Then put a check beside "Enabled". Click "Save", then click "Link to Device".

        The following steps will allow you to change the video quality settings of the camera to a "Constant Bitrate." This is required to prevent the NVR from malfunctioning due to the recording load.

        1. The home page of the camera will appear. If you have no video you may go to "Client Settings" to change the protocol to HTTP. After saving that setting video will appear. Some browsers may report "Path does not exist." By pointing the "Folder" to an existing directory on your hard drive one may avoid this error message. Save the settings and go back to "Home." Video should display properly.
        2. Click "Configuration."
        3. Click "[Advanced Mode]."
        4. Navigate to [Audio and] Video. Newer cameras have a different path, instead click Media > Video.
        5. Click "Video settings for Stream 1."
        6. Change the "Video Quality" to "Constant bit rate:."
        7. The "Bit rate restriction" should be set to "Upper bound."
        8. The "Target bit rate:" is dependent on the NVR.
          1. For NR8201 the throughput maximum is 16Mbps. A proper value for the Target bit rate:" is 4 Mbps. Set it and save the settings.
          2. For NR8301 the throughput maximum is 50Mbps. A proper value for the Target bit rate:" is 6 Mbps. Set it and save the settings.
          3. If the NVR "Monitor" page shows camera views that are slow or sluggish, you may reduce the bitrate on the cameras further or use a faster PC.
    2. Motion Detection Configuration
      (Note: In Section 3 a note describes setting the Recording Policy for motion so the NVR may detect the trigger.)
      1. Go to 'Motion Detection'. Enable it. Click 'New', and type a name for the window.
        1. Here are recommended settings for motion detection:
          1. Do not overlap the camera view's gray border.
          2. Do not overlap one window with another.
      2. 'Percentage' and 'Sensitivity' recommendations:
        1. Typically, for a window taking the whole camera view you want the 'Percentage' set to 1 or 2%.
        2. If the camera is only taking up 1/4 of the main view, you would want to multiply that 'Percentage' by four, making 8% a proper value.
        3. Set to 80%-90% for 'Sensitivity'.
        4. 'Save' the window.

          For more advanced motion detection configurations please contact us for technical support.

  5. Storage Setup

    1. The following steps should be performed for each Hard Disk in the NVR:\
      1. After ensuring the drive is locked into the drive bay navigate to the "Storage" location of the NVR and click "Format" as pictured above.
      2. When the drives are done formatting the NVR will automatically start recording to them.
