Digital zoom on VAST - PiP

Introduction

PiP (Picture-in-Picture) is a display layout commonly used in video surveillance applications. As megapixel-class cameras become more common, PiP allows users to view more clear and detailed images in VAST's LiveClient and Playback.

This document describes PiP as it is implemented as a new feature in VAST version 1.5.

  1. What is PiP?
  2. How to use PiP
  3. PiP advantages

What is PiP?

PiP is an extremely useful feature for video surveillance systems and is implemented in VAST's LiveClient and Playback.

PiP provides an intuitive way for users to simultaneously see a global view and a region of interest (ROI) within that global view. Digital zoom can be used to magnify the ROI and present a detailed view of the video. By drawing on multi-channel simultaneously execution, all logged-in LiveClient and Playback users can activate PiP individually.

How to use PiP

  1. Enabling PiP - There are three ways to enable PiP:
    1. Click within the video and drag to frame an ROI.
    2. Right-click in the video and select "Enable PiP" from the pop-up menu.


    3. Press Ctrl+I - After enabling the PiP feature, a movable global view window and an ROI frame will be displayed.

      The ROI frame can be resized as needed. The ROI frame displays a portion of the global view, which is displayed by default in the bottom right corner, but can be repositioned as desired
  2. Disabling PiP - There are two ways to disable PiP:
    1. Right-click in the video to bring up the contextual pop-up menu, then choose "Enable PiP" if checked to deselect it.


    2. Press "Ctrl+I" again.
  3. Basic PiP Settings - PiP settings allow you to adjust the initial position of the global view window. Select Configuration > Client Settings > PiP Settings to open the window.



    Horizontal: Sets the initial horizontal position of the global view window to be left aligned, middle aligned, or right aligned in the live view window or recorded video playback window. Alternatively, the horizontal position can be set as a percentage value representing the distance from the left border of the live view window or recorded video playback window.
  4. Vertical: Sets the initial vertical position of the global view window to be top aligned, middle aligned, or bottom aligned in the live view window or recorded video playback window. Alternatively, the vertical position can be set as a percentage value representing the distance from the top border of the live view window t or recorded video playback window.
  5. Apply to existing windows: Immediately applies the setting to any currently open windows.
  • Other PiP-related settings
    1. Global View: The global view is the original video scaled down to 160x120. It can be positioned anywhere in the LiveClient or Playback window by dragging. To hide the global view, right-click in the video cell and select "Hide Global View" from the pop-up menu, or press Ctrl+H. The icon will appear in the title bar of the window.


    2. ROI (Region of Interest): The ROI frame can be resized and dragged in any direction in the global view window, offering e-PTZ functionality.
    3. Digital Zoom In: Digital zoom in fills the live view window with a magnified version of the image in the ROI frame. The displayed image will change as the ROI frame is moved or resized, with a zoom value of up to 16x available. You can also easily zoom in and zoom out the ROI frame by rolling the mouse back and force.
    4. Snapshot & Print Zoomed In Image: To take a snapshot of the zoomed image, select Edit > Snapshot Zoomed Image. To print, select Edit > Print Zoomed Image.


  • Multi-touch Mode - VAST also allows the use of multi-touch gestures to zoom in or zoom out the image. Zoom in by placing two fingers on a touch-screen display and sliding them away from each other. Zoom out by sliding them toward each other. The multi-touch is a very powerful and it's a major breakthrough in video surveillance system. It's a function that's been implemented in cell phone field mostly, but VIVOTEK implemented this leading technology for the users.


PiP advantages

  1. Easy to use - This very intuitive interface allows users to simply click in the video cell and drag to frame an area to which digital zoom is applied.
  2. Real-time - PiP allows users to view every state immediately in all channels.
  3. Client-side customization - Each client can define a different ROI in their LiveClient and Playback window.
