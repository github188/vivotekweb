Introduction

VIVOTEK provides the motion detection feature to lower storage requirements by only recording when motion is present. Also, the Privacy Mask is available to mask out desired areas of the video. There are some compatibility issues when setting up the motion detection and privacy mask using Internet Explorer 9.

This document explains how to create and manipulate motion detection and privacy mask boxes when using Internet Explorer 9.

  1. Creating motion detection/privacy mask windows:

    To create a motion detection or privacy mask window, click on new and a small window will appear within the video display.

    When using Internet Explorer 9, sometimes you are unable to move or resize the windows.

  2. Compatibility View
    1. To be able to move and size the motion detection or privacy mask windows you need to select compatibility view.
    2. There are two ways to put Internet Explorer 9 into compatibility view. These are indicated in the above figure with the red circles. Click on either one.
    3. Now you will be able to move and size the motion detection/privacy mask windows.
