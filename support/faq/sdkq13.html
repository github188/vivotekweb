How to add VitaminCtrl Class into Toolbox Panel of Microsoft VS2008

  1. Home
  2. FAQ
  3. SDK Q13

VitaminCtrl is included in High Profile SDK. Please find the information from SDK.

  1. Go Tools > Choose Toolbox Items ...
  2. Go "COM Components" page and checked VitaminCtrl Class. Then click OK.
  3. Then, VitaminCtrl Class will show on Toolbox panel.
