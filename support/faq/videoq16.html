Pixel-Per-Foot Calculations

Introduction

Customers often need to choose an appropriate camera for applications such as license plate recognition (LPR) or face recognition due to the many factors that need to be considered. VIVOTEK has designed a simple tool for calculating the PPF (pixel per foot) value for our cameras. Knowing the PPF can be useful in deciding whether the camera is suitable for a particular application.

Please note that the results produced by this tool are for reference only. True performance should be tested using the camera itself.

Determining PPF

  • What is PPF?
    • PPF stands for pixels per foot and is a way of expressing a camera’s resolution relative to the dimensions of the physical space or objects in a field of view. As an example, in Figure 1.1 the actual width of the physical space in the field of view is 14.2 feet, while the image width is 1280 pixels. 1280 pixels divided by 14.2 feet yields a result of 90 pixels per foot, or 90 PPF. The physical width of the VIVOTEK signboard is 1 foot, while the width in the image is 90 pixels, therefore the camera’s PPF value is also 90.

      If we adjust the lens focal length, we can zoom in on the object while maintaining the same resolution, increasing the PPF. For example, in Figure 1.2 the width of the physical space in the field of view is 8.5 feet, while the image width remains 1280 pixels. 1280 pixels divided by 8.5 feet yields a PPF value of around 151. The PPF of the VIVOTEK signboard is also 151.

  • How High a PPF does an Application Require?
    • Different applications require different PPF levels. Generally speaking, LPR software requires at least 60 PPF, while facial recognition requires at least 80 PPF. Please note that you should still inquire with the vendor of the specific application software you are using about the recommended PPF requirements.
  • Additional Application Information
    • The following information is required for calculating PPF:
      • Model name of the camera
      • Focal length of the lens
      • Camera resolution
      • Distance between the camera and object
