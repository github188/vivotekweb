晶睿通讯今日正式宣布 Milestone 完成专属晶睿通讯的驱动程序

2015.03.04

台北，台湾 - 晶睿通讯今日正式宣布 Milestone 正式完成专属晶睿通讯的驱动程序（Dedicated Driver）并于其 XProtect® Device Pack 7.8 里发表。以晶睿通讯 Web API 为基础，Milestone 推出此专属驱动程序，不仅有效提升其开放式平台 IP 影像管理软件与晶睿通讯网络摄像机整合的效率，更是确保了与软硬件沟通协作的稳定性。网站上「下载区」下载。欲了解更完整详尽的支持产品型号，请至Milestone网站 Supported Hardware 上查询。

长期以来，晶睿通讯致力于提供客户可靠的 IP 监控解决方案，并深知客户在安装 IP 监控系统时，尤其重视系统安装的灵活度和设备扩充时的延展性。有鉴于此，软硬件的无缝整合对于维持产品和服务的竞争力来说是相当重要的一环。Web API 主要是以程序语言的方式呈现，衔接了软件系统不同的组成部分。随着此次 Milestone 开放式平台 IP 影像管理软件与晶睿通讯网络摄像机的整合，未来在多元垂直市场应用的项目合作上，晶睿通讯硬件和 Milestone 软件的协作将更稳定、更有效率且带给客户更高的成本效益。网站上「下载区」下载。欲了解更完整详尽的支持产品型号，请至Milestone网站 Supported Hardware 上查询。

Milestone Systems 成立于 1998 年，是开放式平台 IP影像管理软件 (VMS) 的全球领先供货商。Milestone 的软件功能强大、性能可靠且易于使用，已获得全球数以千计客户们的肯定，并具备与各类型网络摄像机和其他系统整合协作的高度弹性。目前 Milestone 在全球 100 多个国家透过当地经销伙伴提供销售和服务。Milestone 的解决方案有效协助组织管理风险、保障人身和财产安全、优化管理程序并降低营运成本。网站上「下载区」下载。欲了解更完整详尽的支持产品型号，请至Milestone网站 Supported Hardware 上查询。

Milestone Systems 企业营销与策略联盟副总经理 Christian Bohn 表示：「我们十分高兴这次的整合带给 Milestone 通路伙伴和客户们更多新的可能性，屡获奖项肯定的 Milestone 开放式平台影像管理软件此次和晶睿通讯网络摄像机的整合，带给了 Milestone 生态系统合作伙伴极大的益处，此次的整合不仅加速了业务的发展，亦是开放式平台技术发展上一个相当重要的进程。」网站上「下载区」下载。欲了解更完整详尽的支持产品型号，请至Milestone网站 Supported Hardware 上查询。

品牌事业处副总经理顾中威表示：「着眼于软硬件设备整合所带给终端用户的强大综效，晶睿通讯一直致力于与全球领先的影像管理软件平台开发厂商紧密合作。今天十分荣幸 Milestone 为加强与晶睿通讯硬件的整合设计推出了专属晶睿通讯的驱动程序，这次的合作除了再次强调了晶睿通讯对于开发世界级安全监控解决方案的坚定信念之外，更进一步强化了我们与 Milestone 的合作伙伴关系，并为彼此开启了更多合作的可能。」网站上「下载区」下载。欲了解更完整详尽的支持产品型号，请至Milestone网站 Supported Hardware 上查询。
