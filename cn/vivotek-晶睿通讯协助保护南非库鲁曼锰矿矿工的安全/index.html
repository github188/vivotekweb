晶睿通讯 协助保护 南非库鲁曼锰矿矿工的安全

背景

矿场是极具挑战性的环境。矿场营运的地形非常险峻，工作环境也充满危险。矿场中的每一项要素都会影响营运。确保安全是至关重要的事。因此，各项规定与要求都非常严格。为了建构安全的矿工作业环境，利用有效的技术是重点所在。位于库鲁曼（Kuruman）尚未运作的新设锰矿场，最近在整个矿场安装了网络摄像机，以监视建筑工人的安全并确保矿场的安全。这套安全系统必须涵盖矿场开始营运后的需求。

由专业公司 I3S Security Solutions 完成安装的这套系统虽然简单，但是面临几项挑战。除了严苛的作业环境之外，对于每个安装项目的各种规定与审核程序皆非常严格。此外，现场没有任何通讯基础设施。以无线联机解决方案建构的通讯基础设施是从头开始建立。为符合矿场的需求，I3S Security 选择了 RADWIN 点对点与点对多点解决方案搭配晶睿通讯的网络摄像机。I3S Security 的股东 John Gerber 表示：「这座锰矿场位于相当平坦的地形，因此必定会出现一些无线电噪声。透过架设涵盖 12 平方公里范围的无线网络，我们可以有效利用摄像机网络，并且为门禁控制系统提供基础设施。」

解决方案

I3S Security 通过与晶睿通讯当地经销商Miro合作，成功完成通讯基础设施与安全系统的安装。安装期间，Miro 除了提供晶睿通讯的网络摄像机产品之外，同时提供了专业咨询服务与技术专业，使得安装过程非常顺利。在此项目中，共安装超过 150 台各种类型的晶睿通讯高阶网络摄像机，包括子弹型网络摄像机 IP8362、IP8331 及 IP8332、鱼眼固定式网络摄像机 FE8171V、快速球型网络摄像机 SD8321E 与 SD8322E，以及 P/T/Z 网络摄像机 PZ8121。上述摄像机涵盖矿场的所有区域，从建筑群、竖坑、入口旋转闸门及其他所有场所。晶睿通讯摄像机获选的原因包括具备 200 至 500 万像素高分辨率影像录像功能，以及容易与其他设备做整合，例如智能手机、门禁控制系统、保全系统及监视软件。此外，晶睿通讯的摄像机非常适合用于户外环境，主要是由于它具备自动光圈镜头与红外线滤镜，在复杂的照明环境中亦可确保优异的影像质量，这对于确保矿场安全而言非常重要。
