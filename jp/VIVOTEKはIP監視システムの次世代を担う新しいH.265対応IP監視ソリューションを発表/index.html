VIVOTEK は IP 監視システムの次世代を担う 新しい H.265 対応 IP 監視ソリューションを発表

2015.11.11

VIVOTEK(台北・台湾)は新しい H.265/HEVC 監視ソリューション対応の 3 メガピクセル、5 メ ガピクセルネットワークカメラ、NVR(ネットワークビデオレコーダー)、プロフェッショナル向け VMS(ビデオ管理ソフトウェア)を発表します。帯域消費の最適化、ストレージの効率化をより 進めるため、VIVOTEK は、優れた費用対効果を実現する「スマートストリーム II」テクノロジー を導入。このスマートストリーム II と H.265 テクノロジーにより、ユーザーは帯域消費を抑制で き、また従来の H.264 によるシステムと比較して最大 80%までストレージ容量を削減できます。 これらは VIVOTEK の IP 監視システムの新たなマイルストーンとなり、この技術的革新をお客 様と共有してまいります。

「この新しいビデオ圧縮技術 H.265/HEVC は、H.264 の 2 倍の効率です。IP 監視ソリューショ ンのリーディング企業として独自開発のスマートストリーム II と組み合わせて大きな進歩をと げることが目標でした。」とマーケティングコミュニケーション部のティム・スーは述べています。 「VIVOTEK のスマートストリーム II は、監視対象物や監視エリアの解像度を最適化することで、 帯域消費を最大限に効率化するものです。H.265 とスマートストリーム II の組み合わせで最も 効率的なシステムが可能となったのです。」とティム・スーは続けます。「市場の要請に応える ために、フロントエンドにはネットワークカメラ、バックエンドには、ネットワークビデオレコーダ ーとビデオ管理ソフトウェアの包括的な H.265 ソリューションをご提供します。こうした総合ソリ ューションにより、ユーザーは画質を犠牲にすることなく、ネットワーク帯域、ストレージ容量の 効率的な運用を行うことができます。」

VIVOTEK は、まず、多様な形態の 3 メガピクセル、5 メガピクセルの範囲の V-Pro H.265 ネッ トワークカメラシリーズを提供していきます。WDR Pro 及び SNV(Supreme Night Visibility)を 搭載した新しい V-Pro H.265 ネットワークカメラは、どのような明るさの中でも高品質な画像を 得られることを確実なものとします。この比類のない H.265 対応のボックス型ネットワークカメ ラはギガビットイーサネットポート、スナップショットフォーカスを装備し、データの高速転送を実 現し、高速で移動する物体も正確に撮影できます。屋外の過酷な条件や妨害行為に耐えるブ レット型、耐衝撃性ネットワークカメラは、IP66 の防水基準、IK-10 の耐衝撃基準をクリアした 堅牢なケースに格納され、 -50°C から 50°C の幅広い温度条件で動作します。

これらの先進的なネットワークカメラと同時に、VIVOTEK は H.265 互換の NVR として、32 チャ ンネル対応の ND9541、16 チャンネル対応の ND9441 をリリースします。この両機種は H.265 圧縮テクノロジーを完全にサポートし、4 つの HDD トレイで合計最大 24TB のストレージ容量を提供します。これは H.264 の圧縮システムと比較して 30%から 50%の録画容量増となりま す。EZConnect アプリを利用すれば、いつでも、どこからでも、iOS、Android デバイスを使って 監視システムをリモートからモニタできます。最後に、VIVOTEK は新世代の VMS、VAST Version 1.11 をリリースします。この新バージョンは H.265 コーデックとソフトウェアライセンスを サポートしています。ソフトウェアライセンスを含むことにより、もうユーザーは USB ドングルの コストや損傷リスクを心配する必要はありません。つまりユーザーは、32 チャンネル分のライ センスを無償で利用でき、どこへでもすぐに監視システムを移動できます。また、新しい VAST Version 1.11 は、インテリジェント警報管理機能を内蔵しており、セキュリティ監視要員が警報 の状態を記録したり、すぐに検索したりといった効率的な管理が行えます。

* VIVOTEK の H.265 対応の 5 メガピクセルネットワークカメラ製品は、2015 年度の第四四半 期末に提供予定です。
