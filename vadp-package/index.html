IQrity / AxxonSoft

  • AnalyticVideo Tracker

    AnalyticVideo Tracker plug-in provides reliable detection and tracking of moving objects with simultaneous analysis of both object and background movement. It classifies moving objects as people or cars, as well as counts people that move in groups. AnalyticVideo Tracker plug-in provides unattended object detection. It classifies such objects as left / missing. All plug-in metadata contain information about moving and unattended objects is transmitted using ONVIF protocol. AnalyticVideo Tracker plug-in has web interface unified for VIVOTEK cameras. It provides the possibility to customize regions of interest, the expected size of the objects and other settings.

    The solution is fully supported by a video surveillance system, Axxon Next. Using VIVOTEK cameras with integrated AnalyticVideo Tracker by IQrity allows you to free the server resource spent on video analysis (commonly 60-75%) of the overall resource of a video surveillance server), thereby to greatly reduce the overall hardware cost of a designed security system.

    Supported Model (by Project Request):

    MD8531H (FW#MD8531-VVTK-020000.flash.pkg), MD8562/ MD8562D (FW#MD8562-VVTK-020113.flash.pkg)
