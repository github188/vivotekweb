VIVOTEK ND8322P

Experience More Advances Now!

  
 

VIVIOTEK’s 8-CH PoE Embedded NVR, ND8322P, has made great advances in providing users powerful functions with a free upgrade to the newest firmware version. Users can experience these useful functions and enhance their surveillance by purchasing VIVOTEK ND8322P Solution Kits.

 
 

  • Highlight Advances

 
   

EZConnect Notification

Enhanced event notification for 24/7 mobile surveillance. Once the ND8322P has an alarm notification, the EZConnect app will receive the push notification and users can directly playback the video immediately via Android/iOS devices.

  

Smart Stream II

The ND8322P can utilize IP Cameras with VIVOTEK’s Smart Stream II technology, reducing bandwidth and storage consumption by up to 50% more than just adopting H.264, optimizing users’ surveillance cost.

   
 

  • New Value-add Features

 
   

RAID 0, 1

The RAID 0, 1 configuration provides further data security and reliable video storage.

24hr Video Export

The export function can backup for 24hr video for all channels. Secure the video for digital evidence or forensic purpose usage.

Device Pack Upgrade

The ND8322P provides device pack upgrades directly to enable VIVOTEK’s new camera settings. Users can save on installation time.

Fisheye Dewarp 1O8R

The ND8322P provides one more fisheye display mode (1O8R) in Live View and video Playback, allowing users to monitor all angles of video. (1O/1P/1R/1O3R/1O8R)
